# PG CLI Cheat Sheet

## Installation

**Assurez-vous que `pip` est installé :**
```bash
sudo apt update
sudo apt install python3-pip
```

**Installation `pgcli` :**
```bash
sudo pip3 install pgcli
```

**Version :**
```bash
pgcli --version
```

## Initialisation

**Connexion `pgcli` :**
```bash
pgcli -h <hôte> -U <utilisateur> -d <base_de_données>
```
